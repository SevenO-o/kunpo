# 快速开始

## 1. 获取 API Key

登录 [KUNPO API 管理后台](https://llm.ziy.cc)，在「令牌」页面创建一个新的 API Key。

<Note>
  API Key 以 `sk-` 开头，创建后仅显示一次，请妥善保管。
</Note>

## 2. 发送第一个请求

### Gemini 3.1 Flash Lite {#gemini-3-1-flash-lite}

```bash
curl -X POST https://llm.ziy.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-你的密钥" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "google/gemini-3.1-flash-lite",
    "messages": [
      {"role": "user", "content": "你好，请介绍一下你自己"}
    ]
  }'
```

### DeepSeek-R1-0528 {#deepseek-r1-0528}

```bash
curl -X POST https://llm.ziy.cc/v1/chat/completions \
  -H "Authorization: Bearer sk-你的密钥" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "DeepSeek-R1-0528",
    "messages": [
      {"role": "user", "content": "你好，请介绍一下你自己"}
    ]
  }'
```

<CodeGroup>
  ```python Python
  from openai import OpenAI

  client = OpenAI(
      base_url="https://llm.ziy.cc/v1",
      api_key="sk-你的密钥"
  )

  response = client.chat.completions.create(
      model="google/gemini-3.1-flash-lite",
      messages=[
          {"role": "user", "content": "你好，请介绍一下你自己"}
      ]
  )

  print(response.choices[0].message.content)
  ```

  ```javascript Node.js
  import OpenAI from "openai";

  const client = new OpenAI({
      baseURL: "https://llm.ziy.cc/v1",
      apiKey: "sk-你的密钥",
  });

  const response = await client.chat.completions.create({
      model: "google/gemini-3.1-flash-lite",
      messages: [
          { role: "user", content: "你好，请介绍一下你自己" }
      ],
  });

  console.log(response.choices[0].message.content);
  ```
</CodeGroup>

<Note>
  `DeepSeek-R1-0528` 是推理模型，响应中会额外包含 `reasoning_content` 字段，用于返回模型的思考过程；最终回答仍在 `content` 字段中。若 `max_tokens` 设置过小，token 预算可能被思考过程占满，导致 `content` 为空。流式模式下，`delta.reasoning_content` 会先于 `delta.content` 返回。
</Note>

## 3. 查看响应

成功的响应格式如下：

```json
{
  "id": "chatcmpl-xxxxx",
  "object": "chat.completion",
  "model": "DeepSeek-R1-0528",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "你好！我是 DeepSeek-R1..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 12,
    "completion_tokens": 86,
    "total_tokens": 98
  }
}
```

## 使用流式输出

在请求中添加 `"stream": true` 即可启用 SSE 流式响应：

<CodeGroup>
  ```bash cURL
  curl -N -X POST https://llm.ziy.cc/v1/chat/completions \
    -H "Authorization: Bearer sk-你的密钥" \
    -H "Content-Type: application/json" \
    -d '{
      "model": "DeepSeek-R1-0528",
      "stream": true,
      "messages": [
        {"role": "user", "content": "用三句话解释量子计算"}
      ]
    }'
  ```

  ```python Python
  stream = client.chat.completions.create(
      model="DeepSeek-R1-0528",
      messages=[{"role": "user", "content": "用三句话解释量子计算"}],
      stream=True
  )

  for chunk in stream:
      if chunk.choices[0].delta.content:
          print(chunk.choices[0].delta.content, end="")
  ```
</CodeGroup>

|--------|----------|----------|

| xAI | Grok-4-Fast | `x-ai/` |

| 智谱 | GLM-5, GLM-4.7 | 无前缀 |

| MiniMax | MiniMax-M2.7 | 无前缀 |

<Note>
  完整模型列表请访问 [KUNPO API 模型广场](https://llm.ziy.cc)。海外模型使用 `供应商/模型名` 格式，国内模型直接使用模型名。
</Note>
