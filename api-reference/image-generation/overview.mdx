# 图片生成

KUNPO 网关统一封装多款图片生成能力，兼容 OpenAI `images` 接口格式。当前提供 **Nano Banana Pro**、**Nano Banana 2**、**GPT Image 2**、**Midjourney** 四款模型（API 调用代号分别为 `Image-GI`、`Image-GI2`、`Image-GPT2`、`Image-MI`），一个 API Key 即可调用，无需关心上游鉴权细节。

## 支持模型

调用时在 `model` 参数中传入 API 模型名；下表「对应名称」为产品常用称呼，便于识别。

| API 模型名 | 对应名称 | 能力 | 推荐接口 | 典型耗时 |
|------------|----------|------|----------|----------|
| `Image-GI` | Nano Banana Pro | 文生图、图生图（参考图 URL） | [同步](/api-reference/image-generation/synchronous) | 30-60 秒 |
| `Image-GI2` | Nano Banana 2 | 文生图（升级版，画质更佳） | [同步](/api-reference/image-generation/synchronous) | 30-60 秒 |
| `Image-GPT2` | GPT Image 2 | 文生图（高质量） | [异步](/api-reference/image-generation/asynchronous)（同步亦可） | 数分钟 |
| `Image-MI` | Midjourney | 文生图（MJ 风格，一次返回 4 张） | [同步](/api-reference/image-generation/synchronous) | 30-60 秒 |

<Note>
  API 模型名（如 `Image-GI`）为平台统一代号，请求示例与 SDK 调用须使用上表左侧名称；「对应名称」仅作识别参考。
</Note>

### 选择建议

| 场景 | 推荐模型 | 原因 |
|------|----------|------|
| 日常文生图、快速出图 | `Image-GI2` | 速度快、性价比高 |
| 图生图 / 风格迁移 | `Image-GI` | 支持传入参考图 URL |
| 高质量精细出图 | `Image-GPT2` | 画质更好，建议走异步接口 |
| 一次生成多张候选图 | `Image-MI` | 单次请求固定返回 4 张 |

<Note>
  `Image-MI` 仅支持文生图（`model` + `prompt`），`size`、`quality`、`images` 等参数对其无效。按次计费，每次请求固定返回 4 张图片。
</Note>

<Note>
  图生图时，参考图须为**公网可访问的 URL**（推荐使用 KUNPO CDN 上的图片地址），否则上游可能报图片下载错误。
</Note>

## 图片存储

生成的图片自动转存至 KUNPO 图床（COS），返回永久 CDN 链接（`https://kunpoapiimg.ziy.cc/...`），不受上游临时 URL 过期限制。

## 请求参数

### 通用参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|:----:|------|
| `model` | string | **是** | 模型名称：`Image-GI`、`Image-GI2`、`Image-GPT2`、`Image-MI` |
| `prompt` | string | **是** | 图片描述文本 |
| `size` | string | 否 | 图片尺寸。仅控制宽高比，**不保证输出像素与传入值一致**，详见下方映射规则 |
| `quality` | string | 否 | 图片质量。GI 映射为分辨率，GPT 映射为质量等级 |
| `response_format` | string | 否 | 目前仅支持 `url`（默认），传入 `b64_json` 仍返回 URL |
| `images` | string[] | 否 | 参考图 URL 数组，用于图生图 |
| `image` | string | 否 | 单张参考图（URL 或 Base64），系统自动处理上传 |

### GPT 专属参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `background` | string | 背景描述 |
| `output_format` | string | 输出格式，默认 `png` |
| `input_fidelity` | string | `high` 或 `low` |

### `size` 映射规则

<Warning>
  **关于输出尺寸：** `size` 参数仅用于指定**宽高比**，生成的图片实际像素尺寸**不会严格等于**传入值。模型会根据内部算法自适应输出分辨率，最终尺寸以实际返回的图片为准。
</Warning>

| `size` 值 | GI 模型（宽高比） | GPT 模型 |
|-----------|:-----------------:|:--------:|
| `1024x1024` / 空 | 1:1 | 1024x1024 |
| `1536x1024` / `1792x1024` / `1280x720` | 16:9 | — |
| `1024x1536` / `1024x1792` / `720x1280` | 9:16 | 1024x1536 |
| `auto` | — | auto |
| 其它 | 回落 1:1 | 回落 1024x1024 |

### `quality` 映射规则

| `quality` 值 | GI 模型（分辨率） | GPT 模型 |
|-------------|:-----------------:|:--------:|
| `hd` / `high` | 4K | high |
| `medium` / `2k` | 2K | — |
| `low` | 1K | low |
| 默认 / 空 | 1K | medium |

## 计费说明

所有图片生成模型均为**按次计费**，每次请求固定扣费，与输出分辨率无关。

| 模型 | 单价（美元/次） | 约合人民币 | 计费方式 |
|------|:-------------:|:----------:|----------|
| `Image-GI` | $0.18 | ≈ ¥1.3 | 固定按次 |
| `Image-GI2` | $0.09 | ≈ ¥0.65 | 固定按次 |
| `Image-GPT2` | $0.05 | ≈ ¥0.36 | 固定按次 |
| `Image-MI` | $0.069 | ≈ ¥0.5 | 固定按次（一次 4 张） |
