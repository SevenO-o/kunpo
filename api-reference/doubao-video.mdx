# 豆包视频生成

豆包视频生成采用异步任务模式：先提交任务获取 `task_id`，再轮询查询结果。

## 接口地址

```
POST https://llm.ziy.cc/v1/video/generations       # 提交任务
GET  https://llm.ziy.cc/v1/video/generations/{task_id}  # 查询结果
```

---

## 可用模型

| 模型名 | 说明 |
|--------|------|
| `doubao-seedance-2-0-260128` | 高品质模式，画质更佳 |
| `doubao-seedance-2-0-fast-260128` | 快速模式，出图更快 |

## 图生视频 / 多模态生成

通过 `metadata.content` 传入图片、视频、音频参考，支持以下模式：

- **图生视频**：以参考图为首帧/参考风格生成视频
- **视频续写**：以参考视频为基础续写
- **首尾帧控制**：指定视频的起始帧和结束帧

<AccordionGroup>
  <Accordion title="图生视频">

      -H "Authorization: Bearer sk-你的密钥" \
      -H "Content-Type: application/json" \
      -d '{
        "model": "doubao-seedance-2-0-260128",
        "prompt": "placeholder",
        "metadata": {

            {

  <Accordion title="Python 完整示例">
    ```python
    import time
    import requests

    BASE_URL = "https://llm.ziy.cc"
    HEADERS = {"Authorization": "Bearer sk-你的密钥"}

    def submit_video(prompt, image_url=None, ratio="16:9", duration=5):
        """提交视频生成任务，返回 task_id。"""
        metadata = {
            "ratio": ratio,
            "duration": duration,
            "watermark": False,
            "generate_audio": True,
        }

        if image_url:
            metadata["content"] = [
                {"type": "text", "text": prompt},
                {
                    "type": "image_url",
                    "image_url": {"url": image_url},
                    "role": "reference_image",
                },
            ]
            prompt = "placeholder"

        resp = requests.post(
            f"{BASE_URL}/v1/video/generations",
            headers=HEADERS,
            json={
                "model": "doubao-seedance-2-0-260128",
                "prompt": prompt,
                "metadata": metadata,
            },
        )
        resp.raise_for_status()
        return resp.json()["task_id"]


    def poll_video(task_id, interval=5, timeout=300):
        """轮询任务直到完成，返回视频 URL。"""
        start = time.time()
        while time.time() - start < timeout:
            resp = requests.get(
                f"{BASE_URL}/v1/video/generations/{task_id}", headers=HEADERS
            )
            data = resp.json()

            if data["status"] == "completed":
                return data["video_url"]
            elif data["status"] == "failed":
                raise Exception("视频生成失败")

            print(f"状态: {data['status']}，等待 {interval}s...")
            time.sleep(interval)

        raise TimeoutError("轮询超时")


    # 使用示例
    task_id = submit_video(
        prompt="镜头缓慢推进，云雾在山间飘动",
        image_url="https://example.com/mountain.jpg",
        duration=8,
    )
    video_url = poll_video(task_id)
    print(f"视频地址: {video_url}")
    ```

### 响应示例

```json
{
  "task_id": "7489123456789012345",
  "status": "completed",
  "video_url": "https://example.com/output/video.mp4",
  "duration": 5,
  "ratio": "16:9"
}
```

### 任务状态

| 状态 | 说明 |
|------|------|
| `pending` | 排队等待中 |
| `processing` | 生成中 |
| `completed` | 生成完成，可获取 `video_url` |
| `failed` | 生成失败 |

<Tip>
  建议每 5 秒轮询一次，视频生成通常需要 30~120 秒。
</Tip>

## 请求参数说明

<ParamField path="model" type="string" required>
  模型名称：`doubao-seedance-2-0-260128`（高品质）或 `doubao-seedance-2-0-fast-260128`（快速）。
</ParamField>

<ParamField path="prompt" type="string" required>
  视频描述文本。使用 `metadata.content` 多模态时可填 `"placeholder"`。
</ParamField>

<ParamField path="metadata" type="object" required>
  视频生成控制参数。
</ParamField>

<Accordion title="`metadata` 子参数">

<ParamField path="metadata.ratio" type="string">
  视频宽高比：`21:9` / `16:9` / `4:3` / `1:1` / `3:4` / `9:16` / `adaptive`。
</ParamField>

<ParamField path="metadata.duration" type="integer">
  视频时长（秒），范围 4~15。
</ParamField>

<ParamField path="metadata.watermark" type="boolean" default="false">
  是否添加水印。
</ParamField>

<ParamField path="metadata.generate_audio" type="boolean" default="true">
  是否生成背景音频。
</ParamField>

<ParamField path="metadata.tools" type="array">
  启用联网搜索增强（仅纯文本输入时有效）：`[{"type": "web_search"}]`。
</ParamField>

<ParamField path="metadata.content" type="array">
  多模态输入数组，包含 `text` / `image_url` / `video_url` / `audio_url` 元素，每个元素可附加 `role` 字段。

  `role` 可用值：`reference_image` / `reference_video` / `reference_audio` / `first_frame` / `last_frame`。
</ParamField>

</Accordion>
